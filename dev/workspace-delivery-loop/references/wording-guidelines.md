# Wording Guidelines (避免歧义)

## 禁用措辞（改写为可判定）
- “尽量/尽可能” → “必须/在 X 条件下必须”
- “更好/更合理” → “满足 A、B、C 条件”
- “支持大多数情况” → “明确列出支持与不支持的清单”

## 验收标准写法

推荐用 Given/When/Then：
- Given 未配置 X  
  When 进入页面  
  Then 显示提示且按钮禁用

避免：
- “页面表现正常”
- “体验更顺滑”

## 范围与非目标

好：
- “不涉及历史数据迁移”
- “不改变鉴权流程”

差：
- “暂时不考虑迁移”
- “以后可能会做”
