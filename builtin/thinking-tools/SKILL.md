## 目标

提供一套高频思维工具的简明说明与组合方式，帮助你在复杂问题中选对视角、避免偏差、输出可验证结论。

## 使用方式（推荐流程）

1. **定义问题与边界**：一句话说明要解决什么、成功标准是什么。
2. **选择 3-5 个工具**：根据问题类型挑选对应工具，不必全用。
3. **按顺序应用**：从简化/澄清 → 证据/逻辑 → 风险/二阶效应 → 行动。
4. **输出结论与验证计划**：结论必须可验证，并说明不确定性。

## 工具清单（简要介绍）

### 剃刀与原则
- **奥卡姆剃刀**：在解释充分的前提下，优先选择最简单的解释。
- **汉隆剃刀**：能用无知或疏忽解释的，不要归因于恶意。
- **希钦斯剃刀**：没有证据的主张可以被同样无证据地否定。
- **格莱斯剃刀**：人们通常遵循合作原则，优先用善意解释话语。
- **第一性原理**：拆到不可再分的事实与约束，再重建方案。
- **逆向思维（Inversion）**：从“如何失败”或“如何避免”倒推方案。
- **笛卡尔怀疑**：系统性怀疑每个前提，直到留下可靠基点。

### 风险与不确定性
- **预演失败（Pre-mortem）**：假设方案已失败，倒推失败原因。
- **反事实思维**：用“如果不是这样会怎样”检验因果与决策。
- **二阶思维**：评估行动的后续效应与连锁反应。
- **黑天鹅理论**：警惕低概率高影响事件，避免过度自信。
- **蝴蝶效应**：微小变化可能引发巨大系统性影响。
- **临界点（Tipping Point）**：系统在阈值附近会发生质变。

### 概念与系统
- **涌现（Emergence）**：整体行为不能简单由个体性质叠加解释。
- **反馈循环（Feedback Loops）**：正反馈放大、负反馈稳定。
- **瓶颈理论（Theory of Constraints）**：系统受限于最弱环节。
- **杠杆点（Leverage Points）**：小投入撬动大变化的关键位置。
- **熵增定律**：系统自然趋于无序，需持续投入维持秩序。

### 问题诊断
- **五为什么分析法（5 Whys）**：连续追问“为什么”直到找到根因。

### 认知与偏差
- **确认偏误**：倾向寻找支持既有观点的证据。
- **幸存者偏差**：只看成功样本而忽略失败样本。
- **锚定效应**：初始信息影响后续判断。
- **邓宁-克鲁格效应**：能力低时更易高估自己。
- **框架效应**：同一问题因表述不同而改变判断。
- **可得性启发式**：容易想起的例子被过度高估。
- **代表性启发式**：以“典型性”替代真实概率。

### 推理与论证
- **归纳/演绎/溯因推理**：从样本归纳、从规则演绎、从现象推最可能原因。
- **贝叶斯思维**：用新证据更新先验概率。
- **费曼技巧**：用简单语言解释复杂概念以检验理解。

### 决策与经济
- **机会成本**：选择意味着放弃，评估被放弃的收益。
- **沉没成本谬误**：已投入成本不应影响未来决策。
- **边际效用递减**：投入越多，新增收益趋于下降。
- **长尾理论**：低频需求的集合可能具备巨大价值。

### 社会与组织
- **博弈论**：分析多方策略互动与均衡。
- **马太效应**：优势会自我强化，强者愈强。
- **彼得原理**：人在组织内升至不胜任层级。
- **帕金森定律**：工作会膨胀以填满可用时间。
- **旁观者效应**：责任分散导致行动延迟。
- **从众心理**：群体压力影响个体判断。
- **权威偏误**：过度信任权威而忽视证据。
- **破窗理论**：小问题不治理会引发更大失序。
- **康威定律**：系统设计会反映组织沟通结构。

### 思维实验与身份
- **薛定谔的猫**：在观察前处于多种可能状态，提醒认知不确定性。
- **忒修斯之船**：持续替换后是否仍为同一事物，提醒身份与连续性问题。

### 可靠性直觉
- **林迪效应**：存在越久的事物越可能继续存在。
- **墨菲定律**：能出错的地方最终会出错，需要冗余与防护。

## 组合建议（示例）

- **快速判断**：奥卡姆剃刀 → 证据检查（希钦斯） → 二阶思维 → 预演失败  
- **高不确定性**：贝叶斯思维 → 反事实思维 → 黑天鹅 → 预演失败  
- **系统性问题**：反馈循环 → 瓶颈理论 → 杠杆点 → 临界点  
- **组织与协作**：康威定律 → 彼得原理 → 帕金森定律 → 从众/权威偏误  

## 约束规则

- 只选与问题最相关的工具，不做“名词堆砌”。
- 若证据不足，必须标注不确定性并提出补充信息。
